# Find My Nanny UI Style Guide

Goal: Warm, trustworthy, modern marketplace feel. Simple, clean layout with soft neutrals and a calm accent.

## Color Tokens
- Ink: #1F1A17
- Muted Ink: #5F5852
- Canvas: #F7F3EE
- Card: #FFFFFF
- Line: #E7DFD6
- Accent: #2F7D7B
- Accent Dark: #1F5A59
- Accent Soft: #D8ECEB
- Warning: #B45309

## Typography
- Titles: 28–32, weight 700
- Section titles: 18–20, weight 700
- Body: 15–16, weight 400–500
- Meta/helper: 12–13, weight 500, Muted Ink

## Spacing + Radius
- Page padding: 20–24
- Section spacing: 18–24
- Card padding: 16–20
- Input height: 44–48
- Radius: 14–18

## UI Patterns
- Hero area with short headline + benefit line
- Elevated cards for forms
- Accent buttons with clear contrast
- Subtle borders for input fields and cards
